## Getting Started

### Clone the Repository
``` bash
git clone https://github.com/Kapilan1998/Flask-smart-expense-tracker.git
cd Flask-smart-expense-tracker
```

### Install Dependencies
``` bash
pip install -r requirements.txt
```

### Run the Application
``` bash
python app.py
```

### Access the Application
``` bash
http://localhost:5000/
```
